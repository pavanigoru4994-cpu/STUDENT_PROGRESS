# Student Progress Management System
# Project Title

Student Progress Management System

# Project Description

The Student Progress Management System is a Python-based application that helps track and evaluate a student's academic performance.

This project allows users to:

Enter student details

Enter subject marks

Calculate total and average

Check pass/fail status

Display overall performance

It is useful for understanding how student performance can be analyzed using programming.

# Features

Enter student name

Input multiple subject marks

Calculate total marks

Calculate average marks

Display PASS or FAIL status

Show overall progress clearly

Simple and easy-to-use interface

# Technologies Used

Python

VS Code

Command Prompt / Terminal

# How to Run the Project

Open VS Code

Open the project folder

Open Terminal

Run the command:

python progress.py

# Grading Criteria Example

If any subject marks < 35 → FAIL

Average ≥ 75 → Excellent

Average ≥ 60 → Good

Average ≥ 50 → Average

Below 50 → Needs Improvement

# Example Input & Output

Input:
Enter Student Name: Ravi
Enter marks for 5 subjects: 80, 75, 90, 60, 70

Output:
Total Marks: 375
Average: 75.0
Status: PASS
Performance: Excellent
